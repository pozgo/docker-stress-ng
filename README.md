Docker Image with stress-ng tool. This image is acting like `stress-ng` command and passes all arguments.

### Build

    docker build -t polinux/stress-ng .

### Basic Usage

    docker run \
      -ti \
      --rm \
      polinux/stress-ng \
        --cpu 8 \
        --io 4 \
        --vm 2 \
        --vm-bytes 128M \
        --fork 4 \
        --timeout 10s
